## Short Description (for GitHub About)
Android app that automatically checks if your Pokemon Go version matches PGSharp's supported version with scheduling and notifications.

## Features to Highlight
- ✅ Automatic version comparison
- ✅ Background scheduling (every 12 hours)
- ✅ Custom time scheduling (up to 4 times daily)
- ✅ Material Design 3 UI
- ✅ Dark/Light theme support
- ✅ Push notifications
- ✅ Direct download integration
- ✅ Collapsible settings panel
- ✅ No root required
